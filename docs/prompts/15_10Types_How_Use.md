# Prompts

```
당신은 스타트업 PO, PM, CTO 각 50년차 전문가 입니다. 

SWOT 분석을 통해서 아래 문제 상황을 해결해주세요. 


[문제 상황]
최근 앱 출시 후 초기 사용자는 모았으나, 경쟁사 앱의 공격적인 마케팅으로 인해 사용자 증가세가 둔화되었습니다. 다음 분기 마케팅 전략을 새롭게 수립해야 하는 상황입니다.
```

# Output

---

## 🧭 1. SWOT 분석

| 구분                     | 내용                                                                                                                                                         |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Strengths (강점)**     | - 이미 앱을 성공적으로 출시하여 **기초 유저풀 확보**<br>- **핵심 기능과 UX 검증 완료**, 사용자 만족도 일정 수준 유지<br>- 스타트업 특유의 **의사결정 속도 및 개발 민첩성**<br>- **데이터 기반 의사결정 체계** 일부 구축 (앱 분석도구 운영 중) |
| **Weaknesses (약점)**    | - **마케팅 자원 및 예산 제한**으로 브랜드 인지도 낮음<br>- 경쟁사 대비 **광고·프로모션 노출 빈도 부족**<br>- 사용자 리텐션(재방문율)이 낮고 **커뮤니티 효과 미약**<br>- 신규 기능의 **차별화 포인트가 명확히 전달되지 않음**              |
| **Opportunities (기회)** | - 초기 유저 데이터 기반으로 **정교한 타겟 마케팅 가능**<br>- 시장 내 **세분화된 틈새 수요**가 존재<br>- **UGC(사용자 생성 콘텐츠), 리뷰, 추천 시스템** 강화 여지 있음<br>- 경쟁사의 과도한 마케팅으로 일부 사용자들의 **광고 피로감 형성**   |
| **Threats (위협)**       | - 경쟁사의 **공격적 마케팅 및 프로모션** 지속<br>- **대형 플랫폼 진입** 가능성 (카카오·네이버 계열 등)<br>- **사용자 기대치 상승**으로 UX·기능 개선 요구 가속화<br>- **시장 포화 및 이탈률 증가**로 성장세 둔화 위험                |

---

## 🎯 2. SWOT 기반 핵심 전략 방향

| 전략 축                        | 전략 명                      | 구체 실행 방안                                                                                                                          |
| --------------------------- | ------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| **S-O 전략 (강점을 활용해 기회를 확대)** | **데이터 기반 타겟 성장 전략**       | - 초기 사용자 데이터를 분석해 **리텐션 높은 세그먼트** 정의<br>- 이 세그먼트 중심으로 **맞춤형 광고 캠페인 (SNS/커뮤니티 타겟팅)**<br>- 앱 내 사용자 여정 분석을 기반으로 **개인화 추천 알고리즘** 강화   |
| **S-T 전략 (강점으로 위협 대응)**     | **UX 차별화 중심의 방어 전략**      | - 경쟁사 대비 우월한 **UX·UI 개선 로드맵** 수립<br>- **기능보다 사용 경험 중심의 캠페인 메시지** 전환 ("쉽고 빠른 XX 경험")<br>- 경쟁사 과열 마케팅과 달리 **브랜드 신뢰 중심의 브랜딩 캠페인** 진행 |
| **W-O 전략 (약점을 기회로 전환)**     | **커뮤니티·UGC 활성화 전략**       | - 사용자가 콘텐츠·리뷰·사용팁을 공유하도록 **앱 내 커뮤니티 기능** 강화<br>- 추천인 제도·리워드 프로그램으로 **바이럴 확산 유도**<br>- **초기 충성 유저를 브랜드 앰배서더로 육성**                  |
| **W-T 전략 (약점·위협 최소화)**      | **한정 자원 효율화 + 리텐션 강화 전략** | - **마케팅 ROI 중심 KPI 재설정** (DAU보다 LTV·CAC 비율 중심)<br>- **구독형 또는 멤버십 기능**으로 유료 사용자 전환 유도<br>- **이탈 유저 리마케팅 자동화 시스템** 구축               |

## 🧩 3. 다음 분기 실행 로드맵 (예시)

| 단계               | 기간     | 주요 액션                            | 책임 주체      |
| ---------------- | ------ | -------------------------------- | ---------- |
| **1단계: 진단·기획**   | 1~2주   | 사용자 세그먼트 및 리텐션 분석 / 경쟁사 캠페인 벤치마크 | PO·데이터 분석가 |
| **2단계: 전략 수립**   | 3~4주   | 타겟 캠페인 설계, UX 개선안 도출, 커뮤니티 기능 기획 | PM·UX 리드   |
| **3단계: 실행·출시**   | 5~10주  | SNS 리타겟팅 캠페인, 추천인 프로그램 런칭        | 마케팅팀·개발팀   |
| **4단계: 피드백·고도화** | 11~12주 | 데이터 기반 리텐션·CAC 분석, 캠페인 효율 평가     | PO·CTO     |

## 💡 4. 핵심 시사점

> **“확산보다 유지, 광고보다 경험”**
> 다음 분기의 마케팅 전략은 단순 사용자 확대보다 **충성 유저 중심의 데이터 기반 확장 전략**으로 전환해야 합니다.
> 즉, “많은 사람에게 노출”보다 “올바른 사람에게 깊은 인상”이 핵심입니다.
